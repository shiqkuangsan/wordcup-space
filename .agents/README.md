# Project Agent Assets

This directory is part of the repository.

It contains repo-local Codex skills and optional agent tooling instructions so
other users can clone the project and use the same prediction, analysis,
betting-plan, parlay, odds-capture, slip-recording, settlement, and review
workflows inside Codex.

## Tracked

| Path | Purpose |
|---|---|
| `.agents/skills/` | Repo-local Codex skills shipped with the project. |
| `.agents/skills/wordcup-space/` | Umbrella entrypoint for this workspace. |
| `.agents/skills/codex-match-predictor/` | Score prediction workflow. |
| `.agents/skills/codex-betting-operator/` | Betting analysis and operations workflow. |
| `.agents/skills/football-data/` | Football data helper skill. |
| `.agents/skills/betting/` | Odds math helper skill. |
| `.agents/skills/markets/` | Prediction-market helper skill. |

## Ignored Local Runtime

| Path | Reason |
|---|---|
| `.agents/sports-skills-venv/` | Local Python virtualenv generated by `pnpm setup:agents`. |
| `.agents/.venv/` | Alternative local Python virtualenv. |
| `.agents/cache/` | Local cache. |
| `.agents/tmp/` | Local temporary files. |

## Setup

For a normal clone:

```bash
pnpm install
pnpm run setup:local
```

For optional sports-skills CLI runtime:

```bash
pnpm setup:agents
```

Verify the shipped skills:

```bash
pnpm verify:skills
```
